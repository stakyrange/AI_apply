## 项目简介

做饭规划助手是一款基于Python的桌面应用程序，帮助用户规划每周的饮食菜单，自动生成采购清单，并管理个人菜谱库。

### 主要功能

- **周计划管理**：可视化编辑每周的午餐和晚餐菜单
- **智能采购清单**：根据周计划自动生成食材清单，自动合并相同食材
- **菜谱管理**：添加、编辑、删除菜谱，支持自动填充食材信息
- **数据持久化**：所有数据自动保存为JSON格式

## 使用方法

### 启动应用

双击 `cooking_planner.py` 或在命令行运行：
```bash
python cooking_planner.py
```

### 界面说明

应用包含三个主要标签页：

#### 1. 周计划

- 点击"周"选择器切换不同周次（1-4周）
- 点击菜品按钮选择/更换菜品
- 弹出选择器窗口，可搜索菜品
- 菜品自动保存到对应日期

#### 2. 采购清单

- 点击底部"生成采购清单"按钮
- 系统自动汇总当前周所有食材
- 相同食材自动合并，显示使用日期
- 点击"复制清单"可复制到剪贴板

#### 3. 菜谱管理

- 左侧：选择菜谱分类，查看菜谱列表
- 右侧：显示选中菜谱的详细信息
- 添加菜谱：只需输入菜名，系统自动匹配已知菜谱并填充食材
- 编辑/删除：选中菜谱后操作

### 底部按钮

- **保存数据**：手动保存当前所有数据到JSON文件
- **生成采购清单**：根据当前周计划生成采购清单
- **复制清单**：将采购清单复制到剪贴板

## 常见问题

### Q: 添加菜谱时如何自动填充食材？
A: 输入菜名后，系统会自动匹配内置的70+道已知菜谱。如果匹配成功，会自动填充菜系、食材、调味料等信息。

### Q: 如何修改已选菜品的食材信息？
A: 在"菜谱管理"标签页中，选中菜谱后点击"编辑菜谱"，可修改所有信息。

### Q: 采购清单如何导出？
A: 生成清单后点击"复制清单"，然后粘贴到任意文本编辑器或聊天软件中保存。

### Q: 窗口缩小后按钮消失怎么办？
A: 这是已知问题，建议保持窗口在合适大小。底部按钮已优化为固定显示。
